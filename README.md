# Bloc de notas
Esta aplicacion tiene un panel donde navegar por todas las notas escritas y
al clicar sobre una de ellas, dara la opcion de editarla. Al dejar
pulsado, se pueden renombrar las notas o eliminarlas. Las notas se guardan sin que el usuario tenga que hacer nada. 

Las notas se guardan en la ruta predeterminada de Android de la aplicacion.
